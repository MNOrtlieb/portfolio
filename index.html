Unifilar

Unifilar Na modalidade unifilar, os projetos consistem em representações simplificadas do sistema elétrico, mostrando todas as conexões em uma única linha. Estes projetos são ideais para uma visão geral e compreensão rápida do sistema.

São particularmente importantes para evitar confusão com o número de strings e entradas no inversor, garantindo que o projeto seja dimensionado corretamente em termos de corrente máxima, tensão e queda de tensão por distância.

Trifilar

Já na modalidade trifilar, são utilizadas três linhas para representar as conexões elétricas: uma linha para a fase, uma linha para o neutro e uma linha para o terra. Este tipo de representação é mais detalhado e facilita a compreensão das interconexões entre os diferentes componentes do sistema elétrico.

Minha experiência inclui a elaboração de projetos unifilares e trifilares, garantindo precisão e eficiência em cada projeto. Esses projetos são fundamentais para a instalação de sistemas fotovoltaicos, garantindo uma implementação segura e eficiente.
